# BrainCare - 脑电理疗健康应用

## 项目概述

BrainCare是一款专注于脑电理疗的健康应用，提供实时生理数据监测、心理健康评估、理疗课程、记录统计等功能。应用采用React Native开发，支持iOS和Android双平台。

## 项目结构

```
BrainCare/
├── src/
│   ├── components/          # 通用组件
│   │   └── CustomerService.tsx    # 客服组件
│   ├── navigation/          # 导航配置
│   │   └── TabNavigator.tsx       # 底部导航栏
│   ├── screens/             # 页面组件
│   │   ├── HomeScreen.tsx          # 首页
│   │   ├── AssessmentScreen.tsx    # 评估页面
│   │   ├── WellnessScreen.tsx      # 理疗页面
│   │   ├── RecordsScreen.tsx       # 记录页面
│   │   └── ProfileScreen.tsx       # 我的页面
│   └── styles/              # 样式文件
│       └── theme.ts               # 主题配置
├── App.tsx                  # 应用入口
└── package.json            # 依赖配置
```

## 功能特性

### 1. 首页 (Home)
- **动态问候语**: 根据时间自动显示"上午好"、"下午好"、"晚上好"
- **待办事项管理**: 
  - 可编辑的任务列表
  - 完成状态切换
  - 提醒时间设置
- **我的收藏**: 分为"攻略"和"任务"两个标签，展示收藏的课程和计划

### 2. 评估页面 (Assessment)
- **实时脑电数据**: 显示总波幅、Alpha、Beta、Gamma、Delta、Theta等脑电指标
- **实时心电数据**: 心率、血压、血氧监测
- **心理健康量表**: GAD-7焦虑自评、PHQ-9抑郁自评
- **综合健康评分**: 基于多维度数据生成健康评分
- **状态分析**: 专注度、焦虑指数、抑郁指数、压力指数的可视化展示
- **睡眠分析**: 周均睡眠时间统计
- **AI健康建议**: 个性化健康建议生成

### 3. 理疗页面 (Wellness)
- **多标签分类**:
  - 计划: 21天早起计划、高效作息优化等
  - 课程: CBT-I睡眠课程、专注力提升训练等
  - 声聊: Alpha脑波音乐、白噪音助眠等
  - AI助手: 全屏聊天界面，支持历史对话
  - 线下服务: 理疗中心、美容院、心理咨询等
- **付费标识**: VIP标识和价格标签
- **收藏功能**: 一键收藏喜欢的内容

### 4. 记录页面 (Records)
- **时间维度切换**: 支持周、月、年三种统计周期
- **时间导航**: 前后切换不同时间段
- **指标统计**: 焦虑指数、抑郁指数、喝水量等多项指标的柱状图展示
- **睡眠统计**: 特殊的浮动柱状图展示睡眠时间段
- **趋势分析**: 对比上一周期的变化趋势

### 5. 我的页面 (Profile)
- **用户信息**: 头像上传、昵称修改
- **功能列表**:
  - 我的账号: 手机号、微信、QQ绑定管理
  - 会员中心: VIP会员购买
  - 声音和通知: 应用设置管理
  - 更改语言: 多语言支持
  - 订单信息: 购买历史查询
  - 操作指南: 应用使用教程
  - 提交建议: 用户反馈
  - 打赏应用: 支持开发
  - 邀请好友: 推广奖励
  - 服务条款和隐私政策
  - 退出登录

### 6. 客服功能
- **智能客服**: AI助手快速响应
- **人工客服**: 
  - 客服电话直拨
  - 商务微信添加
  - 工作时间提示
- **全局可用**: 在所有主要页面的右上角都有客服入口

## 安装和运行

### 环境要求
- Node.js 16+
- React Native CLI
- iOS: Xcode 14+
- Android: Android Studio, JDK 17+

### 安装依赖
```bash
npm install
```

### iOS运行
```bash
cd ios && pod install && cd ..
npm run ios
```

### Android运行
```bash
npm run android
```
